# Alexa Audio
## Version 0.1

A gulp task for converting and uploading audio for Amazon Alexa.


### Install ffmpeg

Alexa Audio requires ffmpeg to be installed.
```
brew install ffmpeg
```

### Usage

Transcode audio files
```
gulp process-audio
```

Upload audio
```
gulp upload-audio
```

Transcode and upload audio
```
gulp audio
```
